üîç Troubleshooting

If you get "Permission denied":

  • This is normal for /etc/exports - MoxNAS uses development mode
  • The functionality still works through the interface

If mount fails:

  • Check if NFS server is running: systemctl status nfs-server
  • Check firewall: sudo ufw status
  • Verify export exists: showmount -e localhost

If no exports show:

  • Create an NFS share through MoxNAS interface first
  • Check that datasets exist in Storage section
