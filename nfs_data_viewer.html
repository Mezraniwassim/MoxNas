🔍 How to Access This Data

🖱️ Method 1: File Manager (Easiest)

Open your file manager and navigate to:

nfs://192.168.1.109/home/wassim/Documents/MoxNAS/storage_pools/data/datasets/www

Or click "Open xdg-open" when your browser asks.

💻 Method 2: Command Line

Mount the NFS share:

sudo mkdir -p /mnt/moxnas
sudo mount -t nfs 192.168.1.109:/home/wassim/Documents/MoxNAS/storage_pools/data/datasets/www /mnt/moxnas
ls -la /mnt/moxnas/
cat /mnt/moxnas/welcome.txt

🌐 Method 3: From Another Computer

From any Linux/Mac computer on your network:

showmount -e 192.168.1.109
sudo mount -t nfs 192.168.1.109:/home/wassim/Documents/MoxNAS/storage_pools/data/datasets/www /mnt/remote
⚠️ Note: The 404 error you saw earlier is normal! NFS shares can't be accessed directly through web browsers as HTTP content. Use the methods above instead.
